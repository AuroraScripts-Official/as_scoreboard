# 📊 Aurora Scoreboard - Glassmorphism UI (ESX)

Ein modernes, extrem performantes Scoreboard für FiveM (ESX Legacy) im angesagten Glassmorphism-Design (halbtransparent / Blur-Effekt).

## ✨ Features
* **Modernes Design:** Wunderschönes, cleanes UI im Glassmorphism-Stil. Gleitet weich ins Bild.
* **Maximale Performance:** Nutzt das ressourcenschonende `ESX.GetExtendedPlayers()` anstelle veralteter Schleifen (0.00ms idle).
* **Live-Statistiken:** Zeigt Spieler-ID, aktuellen Job, die Gesamtspielerzahl sowie eingeloggte Polizisten und Mediziner.
* **Dynamisch:** Liest automatisch deine maximalen Server-Slots aus der `server.cfg` aus.

## ⚙️ Installation
1. Lade das Repository herunter.
2. Schiebe den Ordner in dein `resources` Verzeichnis (empfohlener Name: `as_scoreboard`).
3. Füge dein eigenes Logo als `logo.png` in den Ordner `html/` ein.
4. Stelle sicher, dass `es_extended` auf deinem Server installiert ist.
5. Füge `ensure as_scoreboard` in deine `server.cfg` ein.

## ⌨️ Bedienung
Standardmäßig öffnest du das Scoreboard, indem du **F10** gedrückt hältst. Lässt du die Taste los, schließt es sich sofort weich wieder. Du kannst die Taste in der `client.lua` anpassen (Standard: Control 57).
